# 🌐 الوصول العالمي للكهرباء في المدارس الابتدائية (1999–2020)

**البنك الدولي | الهدف السابع والهدف الرابع من أهداف التنمية المستدامة | بيانات البنية التحتية التعليمية**

---

# 📌 نبذة عن المشروع

يقدم هذا المشروع تحليلًا استكشافيًا شاملًا (**EDA**) لبيانات نسبة المدارس الابتدائية التي تمتلك إمكانية الوصول إلى الكهرباء في أكثر من **170 دولة واقتصادًا** خلال الفترة من **1999 إلى 2020**.

يعتمد التحليل على بيانات **البنك الدولي – مشروع رأس المال البشري (HCP)**، ويُعد هذا المؤشر عنصرًا محوريًا لفهم:

- جاهزية البنية التحتية التعليمية للتحول الرقمي  
- تحقيق العدالة التعليمية  
- دعم التنمية المستدامة  

كما يبرز المشروع العلاقة الجوهرية بين **الطاقة والتعليم**، ودور الكهرباء كعامل تمكين للتقنيات التعليمية (**EdTech**) خاصة في المناطق الريفية والنامية.

# 🎯 أهداف التقرير والقيمة المضافة

## الأهداف

- تحليل الاتجاهات الزمنية عالميًا  
- مقارنة التقدم بين الدول حسب مستويات الدخل  
- إبراز الفجوات الإقليمية  
- دعم صُنّاع القرار بسياسات قائمة على البيانات  

## القيمة المضافة

- الربط بين التعليم والطاقة وأهداف التنمية المستدامة  
- تحويل البيانات الخام إلى رؤى استراتيجية  
- توفير قاعدة بيانات جاهزة للتحليلات المتقدمة  
- دعم البحث العلمي ولوحات المتابعة  

---

# 📊 أبرز مخرجات التحليل

## 🔍 أهم النتائج

- **قفزات تنموية (Leapfrogging):** دول مثل الهند وساحل العاج سجلت نموًا سريعًا بعد 2010 بفضل الطاقة البديلة  
- **استقرار الدول مرتفعة الدخل:** وصول شبه كامل للكهرباء (~100%)  
- **الكهرباء كمُمكّن رقمي:** ارتباط غير مباشر قوي مع الناتج المحلي  
- **عدم المساواة الإقليمية:** فجوات واضحة في الدول منخفضة الدخل

# 🧩 هيكلية البيانات (Dataset Structure)

يتضمن الملف `schools_electricity_access.csv` الأعمدة التالية:

- `Country Code` — رمز الدولة (ISO-3)  
- `Country Name` — اسم الدولة أو التجمع الاقتصادي  
- `Year` — السنة (1999–2020)  
- `Electricity Access (%)` — نسبة المدارس التي تمتلك كهرباء  
- `Type` — دولة أو تجمّع اقتصادي
